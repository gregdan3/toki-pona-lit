## mi tawa tomo mi

| toki pona                |
| :----------------------- |
| lon open suno pona       |
| lon tenpo pi mun lape,   |
| lon pini pi waso kalama  |
| lon sewi loje a,         |
| lon tenpo pi seli pini   |
| lon ike pi moku pan      |
| nasin mi li kama ike,    |
| mi tawa tomo mi          |
| o awen ala mi            |
| o pona e nasin mi        |
| o ken e wile mi          |
| ni la mi tawa sina a     |
| sina wile e mi lon tomo  |
| sina wile kama e mi      |
| mi kute e toki sina      |
| kama la mi wile kin      |
| kon wawa li tawa e mi    |
| nasin mi li pini a       |
| mi tawa nasin sina (lon) |
| tenpo tawa tomo mi       |
| o awen ala mi            |
| o pona e nasin mi        |
| o ken e wile mi          |
| ni la mi tawa sina a     |
| lon open suno pona       |
| lon tenpo pi mun lape,   |
| lon pini pi waso kalama  |
| mi tawa tomo mi          |
